# Canonical File Naming - RippleLogic v9.6.4

## Public naming rule

Use descriptive canonical filenames without draft/process suffixes.

Avoid public canonical filenames containing:
- review-copy suffixes
- `patched`
- `upload`
- `regen`
- `latest_final`

## Final public canonical names

## Bundle naming rule

ZIP archives are convenience bundles, not canonical authorities. Use `Convenience_Bundle` in ZIP filenames to prevent confusion with manifest-pinned canonical artifacts.
