# Public Release Language

## Safe short announcement

MathGov Core Release 2026.09 is now available as a public Tier 1-3 framework/source release for review, pilot design, and worked-run learning. The package specifies RippleLogic, a decision architecture inside the MathGov framework, using the public cascade RG -> RF -> TRC -> CSV -> RLS: Reality Grounding, Rights Floor, Tail-Risk Constraint, Containment and Structural Viability, and RippleLogic Score.

The release is not empirical validation, legal certification, deployment certification, ProofPack, Tier 4, or autonomous governance authority. It is an open architecture for clearer, more auditable, rights-protecting, tail-risk-aware decision-making.

## Claims to avoid

- "MathGov is proven."
- "RLS makes demonstrably better decisions" before RLS validation results exist.
- "This is legally sufficient."
- "This certifies ethical AI."
- "SGP proves AI sentience."
- "The workbook is a validator."
- "Tier 4 is available."

## Strong claims that are currently safe

- The release defines a structured decision architecture.
- The cascade separates admissibility from residual welfare ranking.
- Rights Floor failures cannot be rescued by downstream scoring.
- Tail-risk failures cannot be averaged away by ordinary benefits.
- CSV checks containment and structural viability before RLS.
- RLS is structurally complete as a scoring specification and ready for validation studies.
- The package is suitable for critique, training, synthetic worked runs, and pilot design.
