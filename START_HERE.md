What This Repository Contains

RippleLogic is the MathGov decision engine: a rights-constrained, tail-risk-aware, containment-aware, ripple-scored decision framework for governance, AI alignment, institutional design, and high-stakes public choices.

The current release uses the five-level method:

Reality Grounding
Rights Floor
Tail-Risk Bound
Containment
Ripple Score

Formal shorthand:

RSG -> NCRC -> TRC -> Containment -> RLS
